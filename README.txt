What this is
- A local whiteboard prototype implementing:
  - Select/move tool
  - Sticky notes (double-click to edit text)
  - Shapes (rectangle, circle, diamond, star, triangle, speech bubble)
  - Flexible curved connectors (connect nodes using Connect tool)
  - Export/Import JSON (Konva stage serialization)

How to run
1. Unzip the package.
2. Run a local server in the folder:
   - Python 3: python -m http.server 8000
3. Open http://localhost:8000 in your browser.
4. Use toolbar on the left to switch tools and create elements.
